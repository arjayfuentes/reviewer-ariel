Installing PrimeNG + Tailwind

Complete setup guide for Angular 19

01
Install PrimeNG v19

Install PrimeNG matching your Angular version. Always match the major version number.

bash 
npm install primeng@19
02
Install PrimeNG themes package

PrimeNG 19 uses a separate themes package. This provides presets like Aura, Lara, and Nora.

bash 
npm install @primeuix/themes
03
Install Tailwind v3

Install Tailwind CSS v3 with its required peer dependencies. Use tailwindcss@3 specifically — v4 has compatibility issues with Angular.

bash 
npm install -D tailwindcss@3 postcss autoprefixer
04
Generate Tailwind config

Generate the Tailwind config file in your project root. This creates tailwind.config.js.

bash 
npx tailwindcss init
05
Update tailwind.config.js

Configure content paths and disable preflight to prevent conflicts with PrimeNG styles.

tailwind.config.js 
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{html,ts}",
    "./node_modules/primeng/**/*.{js,ts}"
  ],
  corePlugins: {
    preflight: false,  // ✅ prevents conflicts with PrimeNG
  },
  theme: {
    extend: {},
  },
  plugins: [],
}
⚠️ preflight: false is critical — without it, Tailwind's CSS reset will strip PrimeNG's default component styles.
06
Update src/styles.css

Add the Tailwind directives to your global stylesheet.

src/styles.css 
/* Tailwind directives */
@tailwind base;
@tailwind components;
@tailwind utilities;
07
Configure PrimeNG in app.config.ts

Register PrimeNG with the Aura theme preset and enable HTTP client and animations.

app.config.ts 
import { ApplicationConfig, provideZoneChangeDetection } from '@angular/core';
import { provideRouter } from '@angular/router';
import { routes } from './app.routes';
import { provideHttpClient } from '@angular/common/http';
import { providePrimeNG } from 'primeng/config';
import Aura from '@primeuix/themes/aura';

export const appConfig: ApplicationConfig = {
  providers: [
    provideHttpClient(),
    provideZoneChangeDetection({ eventCoalescing: true }),
    provideRouter(routes),
    providePrimeNG({
      theme: {
        preset: Aura,
        options: {
          darkModeSelector: '.dark'
        }
      }
    })
  ]
};
08
Test in app.component.ts

Import ButtonModule and add a test template to verify both PrimeNG and Tailwind are working correctly.

app.component.ts 
import { Component } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { ButtonModule } from 'primeng/button';

@Component({
  selector: 'app-root',
  imports: [RouterOutlet, ButtonModule],
  templateUrl: './app.component.html',
  styleUrl: './app.component.css'
})
export class AppComponent {
  title = 'reserba-ui';
}
app.component.html

  </div>

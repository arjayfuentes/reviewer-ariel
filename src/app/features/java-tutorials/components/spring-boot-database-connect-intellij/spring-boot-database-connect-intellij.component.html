Connect Database in IntelliJ

Connect PostgreSQL to your Spring Boot project

01
Click the Database Icon

In IntelliJ IDEA, locate and click the Database icon on the right panel to open the Database tool window.

02
Click +

In the Database panel, click the + button to add a new data source.

03
Click Data Source → PostgreSQL

From the dropdown, click Data Source, search for PostgreSQL and select it.

Data Source → PostgreSQL
04
Enter Connection Details

A new window will pop up. Fill in the connection fields:

Field Value
Port 5436
User postgres
Password password
Database reserba
05
Click Test

Click the Test button to verify the connection. You should see a success message if Docker is running and the credentials are correct.

💡 Make sure Docker Desktop is running and the PostgreSQL container is up before testing.
06
Click Apply

Click Apply to save the data source configuration.

07
Data Source will show

The new PostgreSQL data source will now appear in the Database panel on the right side of IntelliJ.

08
Add database properties in application.properties

Add the following datasource properties to your src/main/resources/application.properties file.

application.properties 
spring.datasource.url=jdbc:postgresql://localhost:5435/reserba

spring.datasource.username=postgres

spring.datasource.password=password
09
Add dependencies in pom.xml

Add the PostgreSQL driver and Spring Data JPA dependencies to your pom.xml file.

pom.xml 
<dependency>
  <groupId>org.postgresql</groupId>
  <artifactId>postgresql</artifactId>
  <scope>runtime</scope>
</dependency>

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>
10
Refresh Maven Dependencies

Click Maven on the right panel, then click Reload All Maven Projects to download and apply the new dependencies.

Maven → Reload All Maven Projects

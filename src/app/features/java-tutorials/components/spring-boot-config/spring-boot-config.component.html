Running Configuration

Set server port via command line configuration instead of application.properties

01
Click the Run Arrow

Locate the run arrow button at the top of IntelliJ IDEA and click the dropdown arrow next to it.

02
Click Edit Configuration

From the dropdown menu, select Edit Configurations... to open the run configuration dialog.

03
Click Modify Options

Inside the run configuration dialog, click Modify options to reveal additional configuration fields.

04
Add Environment Variables

From the modify options menu, select Environment variables to add it to the configuration form.

05
Set SERVER_PORT in Environment Variables

In the environment variables textbox, add the following value:

environment variables 
SERVER_PORT=8282
06
Click Apply and Run

Click Apply to save the configuration, then run the Spring Boot app. The server will start on port 8282.

✅ Your Spring Boot app will now run on http://localhost:8282 instead of the default port 8080.

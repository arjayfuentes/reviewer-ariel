Creating a Spring Boot App

Step-by-step setup guide using IntelliJ IDEA

01
Open New Project

In IntelliJ IDEA, navigate to the menu and open the new project dialog.

File → New → Project
02
Choose Spring Boot Generator

On the left panel under Generators, select Spring Boot.

💡 If you don't see Spring Boot, make sure the Spring Boot plugin is installed in IntelliJ IDEA.
03
Configure Project Settings

Fill in the basic project configuration fields.

Field Value
Name your-project-name
Location choose folder path
Language Java
Build System Gradle - Groovy
04
Add Group, Artifact & Package

Fill in the project identity fields used by the build system.

Field Example
Group com.example
Artifact my-app
Package Name com.example.myapp
05
Choose JDK, Java Version & Packaging

Select the runtime and packaging settings for your project.

Field Recommended
JDK 17 or 21
Java Version 17 or 21
Packaging Jar
💡 Then click Next to proceed to the dependencies screen.
06
Add Dependencies

Search and add the following dependencies using the search bar.

Dependencies
Lombok reduces boilerplate code
Spring Web REST API support
Spring Data JPA database ORM layer
PostgreSQL Driver postgres db connector
Spring Security auth & authorization
Spring Boot DevTools hot reload on save
Flyway Migration db version control
🔍 Use the search bar in the dependencies panel to find and add each one.
07
Click Create

Click the Create button to generate your Spring Boot project. IntelliJ will scaffold the project and download all dependencies automatically.

✅ Your Spring Boot project is ready! You'll see the project structure with src/main/java and build.gradle generated.
